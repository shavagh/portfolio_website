Stock Market Simulator Project

This project stimulates stock price changes. It contains 3 types of stocks, currency, commodity and company stocks. price changes are man automatically every 5 seconds at random to stimulate the stock market. User has the ability to buy and sell stocks and all transactions are displayed. All data of stocks and transactions are saved in a file.


Calculator Project

This project is a simple calculator made to show basic programming skills e.g. procedural programming. user can add, minus, multiply and divide to numbers together to get a result. Use of java swing is used to show GUI implementation of calculator.
